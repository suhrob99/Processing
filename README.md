# Processing
Игра "Набивание шариков"
Правила очень просты:
1)Набивать как можно больше шариков так,чтобы шарик касался нижней плитки
2)При попадании шарика в свободную от плитки зону набираются "Жизни"
3)Как только вы промахнетесь более 3 раз,"змейка" начнет заполнять игровую зону,
что вовлечет потерю внимания и приведет к поражению.
4)Будьте внимательны и хорошей игры!
